BINGON

Application bingo simple, en français.

Télécharger l'application
Si le téléchargement semble bloqué à la fin :
  1. C'est souvent normal — le fichier est déjà téléchargé.
  2. Ouvrez vos Téléchargements (ou l'icône de notification en haut).
  3. Appuyez sur bingon.apk.
  4. Acceptez l'installation si le téléphone le demande.
Si le téléphone dit « Application non installée » :
  1. Ouvrez le Play Store → icône profil → Play Protect → désactivez-le temporairement, puis réessayez.
  2. Vérifiez que vous avez assez d'espace libre sur le téléphone.
  3. Téléchargez à nouveau le fichier (supprimez l'ancien bingon.apk d'abord).
